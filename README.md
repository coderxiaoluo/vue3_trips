# vue3_trips
基于vite 搭建的Vue3+Pinia H5项目

#### 软件架构

1. vue3+vite2
2. Sass
3. Vue Router
4. Pinia
5. 百度地图API 和 vant3组件库


#### 安装教程

```
git clone https://gitee.com/coderxiaoluo/xiaoluo_tip_vue3.git
```



#### 使用说明

```
npm install
```



``` 
npm run dev
```

   通过router配置meta里面传递参数

   pinia状态管理
